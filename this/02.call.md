# 📘 JavaScript `call()`

---

# What is `call()`?

`call()` is a method that immediately invokes a function while explicitly setting the value of `this`.

It also allows arguments to be passed individually.

---

# Syntax

```javascript
functionName.call(thisArg, arg1, arg2, arg3);
```

### Parameters

- **thisArg** → Object that should become `this`
- **arg1, arg2, ...** → Arguments passed individually to the function

---

# Basic Example

```javascript
const person = {
    name: "Ashu",

    greet() {
        console.log(this.name);
    }
};

const another = {
    name: "John"
};

person.greet.call(another);
```

Output

```text
John
```

---

# Why?

Normally

```javascript
person.greet();
```

Internally

```text
this = person
```

With `call()`

```javascript
person.greet.call(another);
```

Internally

```text
this = another
```

So

```javascript
console.log(this.name);
```

becomes

```javascript
console.log(another.name);
```

Output

```text
John
```

---

# `call()` with Arguments

```javascript
const person = {
    greet(city, country) {
        console.log(this.name);
        console.log(city);
        console.log(country);
    }
};

const another = {
    name: "John"
};

person.greet.call(another, "Delhi", "India");
```

Output

```text
John
Delhi
India
```

---

# Function Doesn't Need to Belong to an Object

```javascript
function greet() {
    console.log(this.name);
}

const person = {
    name: "Ashu"
};

greet.call(person);
```

Output

```text
Ashu
```

The function is standalone.

`call()` temporarily makes `person` the value of `this`.

---

# Why Does This Work?

Without `call()`

```javascript
greet();
```

JavaScript decides what `this` should be.

With `call()`

```javascript
greet.call(person);
```

You explicitly tell JavaScript

```text
Use person as this
```

---

# Normal Call vs call()

## Normal Call

```javascript
person.greet();
```

```text
JavaScript decides

↓

this = person
```

---

## call()

```javascript
greet.call(person);
```

```text
You decide

↓

this = person
```

---

# Immediate Invocation

`call()` immediately executes the function.

```javascript
greet.call(person);
```

The function runs immediately.

# Interview Questions

## What is `call()`?

`call()` is a method that immediately invokes a function while explicitly setting the value of `this`.

---

## What is the syntax of `call()`?

```javascript
functionName.call(thisArg, arg1, arg2);
```

---

## What is the first argument of `call()`?

The first argument specifies the object that should become `this`.

---

## How are arguments passed in `call()`?

Arguments are passed **individually**.

```javascript
greet.call(user, "Delhi", "India");
```

---

## Can `call()` be used with standalone functions?

Yes.

```javascript
function greet() {
    console.log(this.name);
}

greet.call(user);
```

The function does not need to belong to the object.

---

## Does `call()` invoke the function immediately?

Yes.

Unlike `bind()`, `call()` executes the function immediately.

`call()` **does not copy the function**.
