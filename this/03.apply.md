# 📘 JavaScript `apply()`

---

# What is `apply()`?

`apply()` is a method that immediately invokes a function while explicitly setting the value of `this`.

Unlike `call()`, arguments are passed as an **array**.

---

# Syntax

```javascript
functionName.apply(thisArg, [arg1, arg2, arg3]);
```

### Parameters

- **thisArg** → Object that should become `this`
- **Second Argument** → Array of arguments

---

# Basic Example

```javascript
const person = {
    name: "Ashu",

    greet(city, country) {
        console.log(this.name);
        console.log(city);
        console.log(country);
    }
};

const another = {
    name: "John"
};

person.greet.apply(another, ["Delhi", "India"]);
```

Output

```text
John
Delhi
India
```

---

# Why?

Normally

```javascript
person.greet();
```

Internally

```text
this = person
```

With `apply()`

```javascript
person.greet.apply(another, ["Delhi", "India"]);
```

Internally

```text
this = another
```

So

```javascript
console.log(this.name);
```

becomes

```javascript
console.log(another.name);
```

Output

```text
John
```

---

# Array of Arguments

```javascript
function sum(a, b) {
    console.log(a + b);
}

const numbers = [10, 20];

sum.apply(null, numbers);
```

Output

```text
30
```

`apply()` automatically spreads the array into individual arguments.

Internally

```javascript
sum(10, 20);
```

---

# call() vs apply()

## call()

Arguments are passed individually.

```javascript
greet.call(user, "Delhi", "India");
```

---

## apply()

Arguments are passed as an array.

```javascript
greet.apply(user, ["Delhi", "India"]);
```

---

# What Happens Here?

```javascript
greet.call(user, ["Delhi", "India"]);
```

Output

```text
city = ["Delhi", "India"]

country = undefined
```

Why?

Because `call()` treats the array as the **first argument**.

It does **not** unpack the array.

---

# Immediate Invocation

Like `call()`, `apply()` executes the function immediately.

```javascript
greet.apply(user, ["Delhi", "India"]);
```

The function runs immediately.

---

# Modern JavaScript

Suppose

```javascript
const args = ["Delhi", "India"];
```

Older approach

```javascript
greet.apply(user, args);
```

Modern approach

```javascript
greet.call(user, ...args);
```

Because of the Spread Operator (`...`), `apply()` is used less frequently in modern JavaScript.
